## Control Flow
- **pengulangan**
  - **for**
    ```
    for(nilaiAwal;kondisi;increment/decrement){
    	aksi;
    }
    ```
  - **do**
  - **while**
    ```
    while(kondisi){
    	aksi;
    }
    ```
- **percabangan**
  - **if**
    ```
    if(kondisi){
    	aksi;
    }
    ```
  - **if else**
  - **if else if**
  - **switch**
    ```
    switch(ekspresi){
    case "nilai 1":
		aksi;
        break;
    default:
		aksi;
        break;
    }
    ```
